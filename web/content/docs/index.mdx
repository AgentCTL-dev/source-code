agentctl is a **Kubernetes control plane** for fleets of contract-conformant AI
agents. You declare an `Agent` or `AgentFleet` with the real CRD group
`agents.x-k8s.io/v1alpha1`; the control plane reconciles it to the right
workload and reports status — with every capability (intelligence, A2A,
coordination, scaling, and the full security model) gated **default-off**.

The reference agent runtime throughout these docs is **agentd**, a ~1.3 MB
static binary that is conformant to the Agent Control Contract (ACC). It is the
*reference* implementation, never a dependency — agentctl depends only on the
contract (the [P0 principle](/docs/contract)).
